Manual do Usuário

Bem-vindo à EKLIPSE!

  • 🔍 Buscar Produtos: Use a barra de pesquisa no topo para encontrar itens rapidamente.
  • 🛒 Adicionar ao Carrinho: Clique em “Adicionar ao Carrinho” nas páginas de produtos para armazenar itens.
  • 💳 Comprar: Use o botão “Comprar” para iniciar o login e finalizar a compra.
  • 📦 Acompanhar Carrinho: Visualize os itens no ícone de carrinho no topo direito.
  • 📸 Galeria e Zoom: Passe o mouse sobre a imagem principal para ativar o zoom. Clique nas miniaturas para trocar imagens.
  • 📄 Informações do Produto: Navegue entre as abas para ver descrição, dados técnicos e avaliações.
  • 👤 Login: Ao clicar em “Comprar”, você será direcionado ao login para acessar sua conta.
  • ✅ Avaliações: Leia opiniões reais de clientes ou deixe a sua ao usar o produto.

Se encontrar algum problema (pagamento, entrega, erros técnicos), relatá-lo é simples: visite nossa Central de Suporte.
